## 🚦 Quick Start Guide 🚦

1. Clone this repository:
   ```bash
   git clone https://github.com/Vannakem2021/car-rental-app.git
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the Expo app:
   ```bash
   npx expo start
   ```

4. Choose your device:
   - Android Emulator 📱
   - iOS Simulator 🍎
   - Expo Go on your phone 📲
   - Web Browser 🌐
